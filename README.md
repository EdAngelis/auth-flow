# Como rodar o projeto

1. Crie um arquivo `.env.local` e coloque nele as variáveis de ambiente do arquivo `.example.env.local`.

3. Instalar dependências:

    ```bash
    npm install
    ```

4. Iniciar projeto:

    ```bash
    npm run dev
    ```
